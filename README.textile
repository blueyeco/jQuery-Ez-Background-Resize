h1. Overview

Slightly modified version of the background library that also allows for a top-offset value provided in the options hash.

See original library for all other info about this plugin.

h2.  Optional Settings

Optional arguments are center, opacity, and top offset.

@$("body").ezBgResize({@
	@img     : "image/bg.jpg",@
	@opacity : .75,@
	@center : false@
	@top : "75px"@
@});@
